# AI Search Chatbot using Endee

## 📌 Overview
This project is a simple AI chatbot built using **Endee vector database**.  
It uses embeddings to perform **semantic search** and return the most relevant answer.

---

## 🚀 Features
- Semantic Search using embeddings
- Fast vector storage with Endee
- Chatbot interaction
- Simple and easy implementation

---

## 🧠 How It Works
1. Dataset is converted into vector embeddings  
2. Embeddings are stored in Endee  
3. User query is converted into embedding  
4. Endee searches similar vectors  
5. Best matching result is returned as response  

---

## 🛠️ Tech Stack
- Python  
- Sentence Transformers  
- Endee (Vector Database)  
- Requests
